## C 的语法规则

1. 源文件中所有函数定义之外可以出现哪些语法元素？
```c
#include <stdio.h>		/* 包含头文件 */

#define N 5			/* 宏定义 */

enum tag1 { TYPE1, TYPE2 };	/* 枚举定义 */

struct tag2 { int x, y; };	/* 结构定义 */

int a;				/* 全局变量定义 */

int foo(void);			/* 函数声明 */

error:				/* 标号 */

int main(void)
{
	return 0;
}
```

2. 函数定义之中可以出现哪些语法元素？
```c
int foo(int arg)

	enum tag1 { TYPE1, TYPE2 };	/* 枚举定义 */
	struct tag2 { int x, y; };	/* 结构定义 */
	error:				/* 标号 */

}
```
